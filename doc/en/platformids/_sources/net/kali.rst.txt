Description

Rolling Version Identifier
^^^^^^^^^^^^^^^^^^^^^^^^^^
*KaliLinux* is one of the few distros which maintain a correct version identifier within
*/etc/os-release*, thus anything works fine.
The tested version scheme is the latest from 2018/2019 on, thusfor the older 
simply assuming these may work too.
